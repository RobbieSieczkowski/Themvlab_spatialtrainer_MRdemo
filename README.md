# Final Scope

The final project scope included the following features:

* Standalone Mixed Reality player  
* Touch controller input  
* Basic UI menu  
  * Credits  
  * Scaffolded Learning Mode  
  * Free Practice Mode  
  * Decorative platonic solid projections  
* Scale Trainer  
  * Scale visualization: live representation of a scale with a full set of beats  
  * Scaffolded Learning Mode: voiceover guide through LMA scale basics  
  * Free Practice Mode: practice sets of scales (Dimensional, Diagonal, Axis, Girdle)  
* VFX indicator to highlight the next spatial pull in scale  
* Sonic & haptic feedback  
* Kinesphere calibration  
* Voiceover from Dr. Jenny Oyallon-Koloski, a Certified Movement Analyst (CMA), as the Laban-Bartineiff Movement Analysis (LBMA) domain expert
